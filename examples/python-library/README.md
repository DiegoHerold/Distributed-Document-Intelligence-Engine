# Python Library Examples

These examples demonstrate the public `eixo` import surface.

They do not implement real PDF, Excel or OCR processing.

Run from an installed environment:

```bash
python examples/python-library/basic_local_engine.py
python examples/python-library/inspect_document.py
python examples/python-library/process_document.py
python examples/python-library/submit_job.py
python examples/python-library/custom_capability.py
```
