# ADR-0003 - Modelo documental canonico

## Status

Aceita.

## Contexto

PDFs, planilhas e outros formatos possuem superficies, elementos e estruturas diferentes.

## Decisao

Usar um modelo documental canonico como representacao intermediaria comum, preservando diferencas de cada formato por meio de superficies, elementos, relacoes, evidencias e proveniencia.

## Consequencias

- Templates e schemas operam sobre o modelo canonico.
- Capabilities diferentes podem colaborar sobre a mesma representacao.
- Excel nao deve ser forçado artificialmente ao modelo de paginas; abas podem ser superficies.
